Unity Player installation fails

+

Please follow these steps to resolve the issue:

  • Windows:
    1. Close all web browser windows.
    2. Open Control Panel.
    3. Click on Add or Remove Programs
    4. Right click on Unity Web Player
    5. Click Uninstall.
    6. Reboot your machine.
    7. Download the latest version of the Unity Web Player plugin from: http://unity3d.com/webplayer/
    8. Close all browser windows.
    9. Run the Unity Web Player installer.
  • MacOS
    1. Log onto MacOS with a user account that has admin privileges.
    2. Close all web browser windows.
    3. Delete the file: /Library/Internet Plug-Ins/Unity Web Player.plugin
    4. Empty the trash.
    5. Reboot and log in with an admin account.
    6. Download a new copy of the latest web player plugin from: http://unity3d.com/webplayer/
    7. Close all browsers.
    8. Run the web player installer.

If this does not resolve the issue, make sure Unity is not installed as a plugin in your web browser, by following the instructions below:

Firefox:

Open the browser and click the Tools button -> Extensions -> Plugins Tab (In the newest Firefox you can choose “Extensions” directly).

If the Unity Web Player is listed, please deactivate it, and repeat the steps provided in the top of this guide.

I have installed Unity but can only see a black screen

+

This issue is usually caused by an unsupported browser. Please use a modern browser, such as Firefox. To test if your system is capable of running Unity 3D, please run the following demo.

http://unity3d.com/gallery/demos/live-demos

If you can play the demo but still cannot play Labster, please contact our support through the feedback button located at the bottom of the Labster website (Contact Us page). We will then contact you ASAP to help fix the issue.

Unity Player fails to update

+

There exist multiple solutions to this problem – see below.

A) Try installing the full installer

Sometimes the Unity web player update fails with the lite version, which is offered by Unity as the default install option. Resolve this by downloading the full installer on http://unity3d.com/webplayer.

Note: Be sure to run the installer as an administrator (Right click the downloaded file, and select OK when prompted)

B) Disable all browser addons :

Some browser addons, download managers in particular, have been reported to cause problems with updating unity web player. To avoid this, please disable the addons for the time of the update. Follow the links below to find out how to do this.

  • Firefox: https://support.mozilla.org/fi/kb/disable-or-remove-add-ons
  • Chrome: https://support.google.com/chrome/answer/187443?hl=en )

C) Disable encryption and other factors that limit access to the unity folders:

If you’re using encryption on your drives or other things that could prevent unity installer from accessing the installation directories, please disable them.

To make sure you have necessary access privileges, run the installer as an administrator.

D) Clear Unity cache:

Unity cache does not get emptied when updating, and old cache files can interfere with a newer game version.

Go here and click “remove all” : http://webplayer.unity3d.com/setup/

E) Clear Browser cache

Clear your browser cache in order to update Unity web player, as per the guide below:

http://www.wikihow.com/Clear-Your-Browser%27s-Cache

F) Uninstall and Re-install the Unity web player manually

It’s crucial that these steps are followed – the update corruption has been reported to cause the unity update failures due to multiple reasons.

Follow the below steps closely:

  • Windows:
    1. Close all browsers.
    2. Use Control Panel to uninstall the Unity Web Player.
    3. Reboot your machine.
    4. Make sure the following temp folders are deleted : Win7 and WinXP: %TEMP%\UnityWebPlayer
      Win7: %TEMP%\..\..\LocalLow\unity
      WinXP: %temp%\..\Application Data\Unity
    5. Make sure Unity web player is not enabled as a plugin in your browser (disable or remove if it is)
    6. Download a new copy of the latest web player plugin from: http://unity3d.com/webplayer/
    7. Again close all browsers.
    8. Run the web player installer.
  • MacOS
    1. Log onto MacOS with a user account that has admin privileges.
    2. Close all web browsers.
    3. Delete the file: /Library/Internet Plug-Ins/Unity Web Player.plugin
    4. Empty the trash.
    5. Reboot and log in with an admin account.
    6. Make sure Unity is not enabled as a plugin in your browser – if is, disable / remove
    7. Download a new copy of the latest web player plugin from: http://unity3d.com/webplayer/
    8. Again close all browsers.
    9. Run the web player installer.

G) Enable read and write for admin in Mac

  1. Open Finder > (your hard drive) > Library > Internet Plug-Ins > see Unity Web Player.plugin
  2. Right click and “Show Package Contents”
  3. Browse to Contents > Frameworks
  4. Right click Frameworks and Get Info
  5. Click “Sharing & Permissions” at bottom of the info box, you should see system, staff, everyone.
  6. Unlock with lock icon in bottom right, enter your password
  7. Hit + button bottom left, and highlight Users & Groups > Administrator – and hit “Select” bottom right
  8. You should see this new line item in the custom access list. Might take a few tries to make it appear
  9. When it shows up, change the admin permission to “Read & Write”. (You can also remove Staff if you like with the “-” button)
  10. Close the info box
  11. Run Labster

My browser keeps telling me to install unity web player.

+

Please make sure you have Java and Cookies enabled in your web browser, and that your web browser security level is set to normal or low. After that, go to your control panel and see your installed programs, if you can see Unity web player there, please remove it, then try installing Unity again.

I am shown an “Asset not found” warning message

+

This usually happens when Labster is missing some element that is necessary to load the game. Please refresh your web browser and to reload Labster. If you still encounter the same message, try to clear your web browser cache and check your internet connections.

A button, image, or other element in the game is missing

+

We continually improve our virtual labs, which in odd instances may prevent loading of elements. Try to clear your cache and restart the lab. If the issue persist, please contact our support team.

I'm stuck at [some part] of [some lab], how do I continue?

+

If you get stuck at some point in the lab, please make sure to refer to the theory section of the particular lab - the solution is often there.

The "View theory" button only shows a black screen

+

To resolve this issue, deactivate any pop up blocker, firewall, and anti virus. After that, reload the virtual lab, and test that the “View theory” button works. Contact support if the issue persists.

The Continue button on my personal Labster dashboard is not working

+

Labster uses a saving system that is based on checkpoints, that autosaves your progress once every few questions you answer. Once you reach a certain checkpoint, you can always continue from that point onwards, by clicking continue on the Labster Dashboard.

If you have not reached a new checkpoint, you will always start from the most recent one (or from the beginning if you have not reached any checkpoint).

Always make sure you have reached a new checkpoint before quitting the lab. A checkpoint is indicated by a green “saving game” text element that is flashing on the lower left of the Labster game window.
